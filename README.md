# 🚀 Sales Forecasting System (AI + ML + Dashboard)

A full-stack **Sales Forecasting Web Application** built using **Flask, Machine Learning, and Time Series Analysis (ARIMA)**.  
It predicts future sales and visualizes data with interactive charts.

---

## 🔥 Features

- 🔐 User Authentication (Login/Register)
- 🔑 Forgot Password (OTP-based)
- 🤖 Machine Learning Prediction (Random Forest)
- 📈 Time Series Forecasting (ARIMA)
- 📊 Interactive Charts (Chart.js)
- 📉 ML vs ARIMA Comparison
- 🎨 Stylish Dashboard UI

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript (Chart.js)
- Backend: Flask (Python)
- ML Model: Scikit-learn (Random Forest)
- Time Series: Statsmodels (ARIMA)
- Database: SQLite

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository
git clone https://github.com/iharendra13/sales-forecast.git  
cd sales-forecast

### 2️⃣ Install Dependencies
pip install flask pandas numpy scikit-learn statsmodels

### 3️⃣ Train Model
python train_model.py

### 4️⃣ Run Application
python app.py

### 5️⃣ Open in Browser
http://127.0.0.1:5000/
